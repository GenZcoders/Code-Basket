RODUCTIVITY TOOLS FOR WEB DEVELOPERS

Posted on -14 Feb 2022

  • GITHUB COPILOT

    Copilot is currently had Technical Preview only. This copilot is a powerful tool with its autocompletion and code suggestions - no more wasting time in docs, no more copying from StackOverFlow.

    https://copilot.github.com/

  • FRONT-END CHECKLIST

    • Best productivity tool for front-end devs. Before deploying your website go through this checklist.

    https://frontendchecklist.io/

  • WAKATIME

    • Some people don’t care about how they are spending their coding time but it helps to understand how you are utilizing your time like what projects you are mostly focusing on, what languages you are spending more time on.

    https://wakatime.com/

  • HOPPSCOTCH

    • HoppScotch is an open-source alternative to the postman. API testing is a crucial part of building RESTAPI. You can test API’s with your browser but you will also realize how hard it will be if you’re testing a big API.

    https://www.hopscotch.in/

  • LAMBDATEST

    • You made a website and you want to test it in different browsers because some browsers don’t support some functionalities.

    https://www.lambdatest.com/

  • ANIMATOR BY HAIKU

    • Haiku Animator - a desgin tool for creating UI animatons, Lottie files and interactive web components.

    https://www.haikuanimator.com/

  • COLD TURKEY

    Using cold turkey you can proactively restrict your access to specific desktop appliactions, websites or even your access to the internet in general. Yoou set your own schedule and daily limits for access, based on areas of weaknesses.

    https://getcoldturkey.com/

  • 7 PACE

    7pace timetracker work in Agile, scrum or waterfall environment. Individuals using the tool experience an average of just three seconds to log each task.

    Even if you use GitHub project boards manage your projects 7pace has a beta offering to integreate their time tracking solution right in.

    https://www.7pace.com/

  • F.LUX

    f.lux makes your computer screen look like the room you’re in all the time. When the sun sets it makes your computer look like indoor lights.

    https://justgetflux.com/
